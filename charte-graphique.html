Règles

À faire

  • ✓ Fichiers logo officiels uniquement
  • ✓ Une couleur de division par section
  • ✓ Montserrat (titres), Inter (texte), JetBrains Mono (code)
  • ✓ Contraste ≥ 4.5:1 (AA) minimum
  • ✓ Mobile-first, tester sur mobile en priorité
  • ✓ Dark mode sur chaque composant
  • ✓ Self-hosted : polices, icônes, images (zéro CDN)
  • ✓ Unités rem/em pour le texte, 16px minimum
  • ✓ Radius 16px (cartes), 12px (boutons), 8px (champs)
  • ✓ Conteneur max 1280px, centré
  • ✓ Respecter prefers-reduced-motion
  • ✓ Glassmorphism : opacité ≥ 70% si texte
  • ✓ Zones de tap minimum 44×44px (Apple HIG)
  • ✓ Ordre de focus logique (tab order = ordre visuel)
  • ✓ 5 états interactifs : default, hover, focus, active, disabled

À éviter

  • ✗ Déformer, recadrer ou recolorer les logos
  • ✗ Mélanger plusieurs couleurs de divisions
  • ✗ Ombres franches, effets 3D, Liquid Glass
  • ✗ Polices non autorisées (cursive, fantaisie)
  • ✗ Pixels fixes (px) pour les tailles de texte
  • ✗ CDN externes (Google Fonts, cdnjs, etc.)
  • ✗ Texte blanc sur orange/lime
  • ✗ Transparence < 50% avec texte
  • ✗ Capitales sur textes longs
  • ✗ Tutoyer dans les contenus
  • ✗ Animations sans prefers-reduced-motion

Principes

1. Clarté

Fonds clairs, aéré, une couleur par section

2. Subtilité

Ombres douces, animations légères

3. Cohérence

Mêmes tokens, labels, composants partout

4. Accessibilité

AA minimum, AAA visé

5. Souveraineté

Zéro CDN, infrastructure suisse

6. Mobile-first

Mobile d'abord, enrichir pour desktop

Animations : subtiles uniquement (fade-in, hover lift, zoom image). Obligatoire dans chaque projet :

@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
