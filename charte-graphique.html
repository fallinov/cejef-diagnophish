Règles d'utilisation des logos

  • • Zone de protection : hauteur d'une barre verticale autour du logo
  • • Ne jamais déformer, recadrer ou recolorer les logos
  • • Utiliser les fichiers SVG pour le web, PNG HD pour l'impression
  • • Ne pas placer le logo sur un fond trop chargé
  • • Ne jamais recréer les logos — toujours utiliser les fichiers officiels

Règles d'utilisation

À faire

  • ✓ Utiliser les fichiers logo officiels (SVG ou PNG)
  • ✓ Respecter la zone de protection autour du logo
  • ✓ Utiliser une seule couleur de division par page
  • ✓ Privilégier les fonds clairs (#FFFFFF ou #F5F5F5)
  • ✓ Appliquer la couleur d'accent sur les boutons, liens et tags
  • ✓ Utiliser Montserrat pour les titres, Inter pour le texte
  • ✓ Garantir un contraste ≥ 4.5:1 (AA) ou ≥ 7:1 (AAA)
  • ✓ Tester sur mobile en priorité (mobile-first)
  • ✓ Utiliser les unités rem/em pour les tailles de texte

À éviter

  • ✗ Déformer, recadrer ou recolorer les logos
  • ✗ Placer le logo sur un fond trop chargé
  • ✗ Mélanger plusieurs couleurs de divisions
  • ✗ Ajouter des dégradés, ombres portées ou effets 3D
  • ✗ Utiliser des polices non prévues dans la charte
  • ✗ Afficher le logo en dessous de la taille minimale
  • ✗ Utiliser des pixels fixes (px) pour les tailles de texte
  • ✗ Ignorer les tests d'accessibilité
  • ✗ Utiliser les capitales sur des textes longs
